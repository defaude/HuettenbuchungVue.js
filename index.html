What exactly is happening here?

As soon as your Vue app starts, it will look for the element with the id "huettenbuchung", delete its contentsand assume control over that element. The whole bundling, precompiling, etc. is all done by steal.js. Whenever you like your current dev state, just stop your http server and run npm run build. This will bundle all the stuff into the "dist" folder. Then, basically just copy the whole dist folder to your web server and include dist/bundles/HuettenbuchungVue.js/index.js in a page where the "huettenbuchung" id is given to an element. (see prod-demo.html for an example.
